gist-it  •  Embed files from a github repository like a gist

Description

gist-it lets you take a file from your GitHub repository and embed it into any webpage, just like a gist

Syntax highlighting by google-code-prettify

Usage

To embed a file, wrap a GitHub url in a <script> tag like the following:

<script src="{{ dispatch.url_for() }}/github/$user/$repository/raw/$branch/$path"></script>
